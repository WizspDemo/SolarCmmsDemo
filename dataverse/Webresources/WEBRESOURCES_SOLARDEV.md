## 4. Λήψη περιεχομένου (download) – MCP dataverse-webresources

Το MCP **user-dataverse-webresources** εκθέτει μόνο τα εξής εργαλεία: **list_web_resources**, **upload_web_resource**, **update_web_resource**, **delete_web_resource**, **publish_all**.  
**Δεν υπάρχει εργαλείο λήψης** (get_web_resource, download_web_resource κ.λπ.), οπότε η κατέβαση των αρχείων **δεν είναι δυνατή μέσω αυτού του MCP**.

Για να κατεβάσεις τα πραγματικά αρχεία στο `dataverse/Webresources/` χρησιμοποίησε ένα από τα εξής:

1. **Power Platform CLI (pac)**  
   - Export: `pac solution export --name Solardev --path . --managed`  
   - Άνοιξε το exported zip και πάρε τα αρχεία από το φάκελο `WebResources/` → αντιγραφή στο `SolarCmmsDemo/dataverse/Webresources/`.

2. **Solution Packager**  
   - Unpack το solution zip με [PowerApps-Solution-Builder](https://github.com/microsoft/PowerApps-Solution-Builder) (SolutionPackager) για ξεχωριστά αρχεία web resources.

3. **Dataverse Web API**  
   - Retrieve κάθε `webresource` με `$select=content,name`· το `content` είναι base64. Decode και αποθήκευσε με extension ανά τύπο (`.html`, `.js`, `.xml`).

Αν στο MCP προστεθεί εργαλείο **get_web_resource** / **download_web_resource**, η λήψη θα μπορεί να γίνει απευθείας από εκεί.
