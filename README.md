# SolarCmmsDemo

Solar CMMS Demo – Dataverse scripts, dashboards, templates and schema for solar plant maintenance.

## Contents

- **dataverse/** – Scripts (Dashboard PowerDashboard, ParkDashboard, ManagerDashboard, ClientDashboard), Forms, Views, schema (DBML, JSON), Phase 1/2 configs
- **Dataverse_Templates/** – Reusable templates (e.g. Entity Team Members Pane)
- **scripts/** – PowerShell (phase2 tables, maintenance team category, export metadata)
- **Sharepoint/** – SharePoint schema and list exports (PvTicketing, SolarMaintenance)
- **Schema/** – ERD, solution plan, SQL/PostgreSQL/DBML schemas

## Docs

- `scripts/README-phase2-solardev-mcp.md` – Phase 2 tables with Dataverse MCP
- `Sharepoint/README_Structure.md` – Sharepoint folder structure
