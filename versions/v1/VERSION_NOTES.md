# Έκδοση v1 – Snapshot Dataverse (SolarCmmsDemo)

**Ημερομηνία:** 2025-03-02

## Τι κάνει αυτή η έκδοση

Αποθήκευση της τρέχουσας κατάστασης του φακέλου `dataverse` του SolarCmmsDemo. Περιλαμβάνει scripts για φόρμες (Preventive Task, Corrective Work Order, New Warehouse To Location), view scripts, dashboards (Park, Power, Manager, Client) και schema/sync αρχεία. Χρησιμεύει ως σημείο αναφοράς για να αναπαραχθεί ή να συγκριθεί η συγκεκριμένη κατάσταση του κώδικα.

## Αλλαγές από v0

- Πρώτη έκδοση: snapshot του `SolarCmmsDemo/dataverse` (Forms, View, Dashboard, schema).
